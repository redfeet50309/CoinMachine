GitHub Personal Access Token

設定 PAT 後，在主頁新增/移除股票會自動 commit 到 repo，下次 Actions 跑批時就會抓新代號的資料。 PAT 只存在你這台電腦的 localStorage，沒有送到任何伺服器。

  1. 到 GitHub 設定 → Fine-grained PAT，按「Generate new token」
  2. 名稱填例如 CoinMachine-watchlist，到期日建議 90 天
  3. Resource owner 選你自己 (redfeet50309)
  4. Repository access 選「Only select repositories」→ 勾 CoinMachine
  5. Permissions → Repository permissions → Contents: Read and write，其餘留 No access
  6. Generate token，複製貼到下面
